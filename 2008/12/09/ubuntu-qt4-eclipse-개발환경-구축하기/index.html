ubuntu-qt4-eclipse-개발환경-구축하기

Dec 9, 2008

시냅틱으로 Qt4 관련 패키지 모두 설치한다.

이클립스 Qt 플러그인을 받는다.
http://trolltech.com/developer/eclipse-integration

Window -> Preference -> Qt -> Add 를 선택하고…
Version Name : Qt4
Bin Path : /usr/bin
Include Path : /usr/include/qt4

프로젝트를 생성해 신나게 코딩한다. ㅋ
