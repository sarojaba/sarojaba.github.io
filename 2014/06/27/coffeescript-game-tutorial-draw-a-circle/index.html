coffeescript-game-tutorial-draw-a-circle

Jun 27, 2014

캔버스에 동그라미를 그리는 코드이다.

window.onload = ->

    # get a reference to the canvas
    canvas = document.getElementById("canvas")
    ctx = canvas.getContext("2d")

    # draw a circle
    ctx.beginPath()
    ctx.arc(75, 75, 10, 0, Math.PI * 2, true)
    ctx.closePath()
    ctx.fill()
