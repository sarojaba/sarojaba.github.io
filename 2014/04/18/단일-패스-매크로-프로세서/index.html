단일-패스-매크로-프로세서

Apr 18, 2014

과제 목표

단일 패스 매크로 프로세서를 구현

프로그램 설명

텍스트 파일을 입력으로 받아 매크로 구문을 해석 후 확장된 코드로 변환시켜주는 단일 패스 매크로 프로세서를 자바로 구현함.

입력 텍스트

L
MACRO
ADD1    &A
L    &A
A    &A
ST    &A
MEND
L
MACRO
ADDS    &A1, &A2, &A3
ADD1    &A1
ADD1    &A2
ADD1    &A3
MEND
ADDS    DATA1, DATA2, DATA3
END
